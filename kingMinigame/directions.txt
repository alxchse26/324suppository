Step 1: Check Your System Architecture
Before downloading, confirm whether your Windows is 64-bit (almost certainly yes on any modern PC):

Press Windows key + R, type cmd, and press Enter to open a Command Prompt.
In the black terminal window, type:

   systeminfo | findstr /C:"System Type"

Press Enter. You'll see output like System Type: x64-based PC. This confirms you need the x64 installer.


Step 2: Download RubyInstaller

Open your web browser and navigate to:
https://rubyinstaller.org/downloads/
On that page, you'll see two sections: "WITH DEVKIT" (at the top, highlighted in bold) and "WITHOUT DEVKIT." If you're getting started with Ruby, use the Ruby+Devkit 3.4.X (x64) installer — it provides the largest number of compatible gems and installs the MSYS2 Devkit alongside Ruby so that gems with C extensions can be compiled immediately. RubyInstaller
Click the bold/highlighted link, e.g. Ruby+Devkit 3.4.X-X (x64). Your browser will download a file named something like rubyinstaller-devkit-3.4.1-2-x64.exe to your Downloads folder.


Step 3: Run the Installer

Open File Explorer (Windows key + E) and navigate to your Downloads folder.
Double-click the file rubyinstaller-devkit-3.4.X-X-x64.exe.
Windows may show a "Microsoft Defender SmartScreen" warning popup. Click "More info", then click the "Run anyway" button that appears. GitHub
If Windows asks "Do you want to allow this app to make changes to your device?", click Yes.


Step 4: License Agreement Screen

The installer opens to a Setup - Ruby window with the title bar reading "License Agreement."
Read (or scroll through) the BSD-style license text.
Select the radio button "I accept the License".
Click the Next > button in the bottom-right corner.


Step 5: Installation Destination & Optional Tasks
You'll see a screen titled "Installation Destination and Optional Tasks."

Destination folder: The default path will be something like C:\Ruby34-x64. You can leave this as-is unless you have a specific reason to change it. If you want to change it, click Browse... and navigate to your preferred folder.
Checkboxes — make sure these are checked:

Add Ruby executables to your PATH ← This is critical. Without it, Ruby commands won't work in the terminal.
Associate .rb and .rbw files with this Ruby installation


Click Install (bottom-right).


Step 6: Wait for File Extraction
A progress bar screen titled "Installing" will appear and files will extract automatically. This typically takes 1–3 minutes. No action needed — just wait.

Step 7: The Final Installer Screen — Launch MSYS2
When extraction finishes, a screen titled "Completing the Ruby Setup Wizard" appears.

Ensure the checkbox "Run 'ridk install' to set up MSYS2 and development toolchain" is checked, then click Finish. GitHub

This will open a new black Command Prompt window automatically.

Step 9: Verify the Installation
Now confirm Ruby is working correctly:

Close any Command Prompt windows that are open from the installer. This is important — the PATH update from Step 5 won't be recognized in windows that were already open.
Open a fresh Command Prompt: Press Windows key, type cmd, and press Enter.
Type the following and press Enter:

   ruby -v

You should see output like:

   ruby 3.4.1 (2025-02-26 revision d2930cf0b6) [x64-mingw-ucrt]
If you see this, Ruby is installed correctly. ✅

Also verify the gem package manager is working:

   gem -v
You should see a version number like 3.6.2.

Step 10: Update RubyGems (Recommended)
The version of Gem that ships with the installer may be out of date. To update it, type: GitHub
gem update --system
Press Enter and wait for the update to complete.

Troubleshooting: "ruby is not recognized"
If Step 9 gives you an error like 'ruby' is not recognized as an internal or external command, your PATH was not set correctly. Fix it manually:

Press Windows key, type environment variables, and click "Edit the system environment variables" from the results.
In the System Properties window that opens, click the "Environment Variables..." button near the bottom-right.
In the "System variables" section (bottom half of the window), scroll down to find Path, click it once to select it, then click Edit....
In the Edit Environment Variable window, click New and add:

   C:\Ruby34-x64\bin
(Adjust the folder name to match whichever version you installed — check C:\ in File Explorer to see the exact folder name.)
5. Click OK on all three open windows to close them.
6. Open a new Command Prompt and retry ruby -v.

You're Done!
Ruby is now installed. You can write .rb files in any text editor and run them with:
ruby your_file_name.rb
To install additional libraries (called gems), use:
gem install <gemname>
For example: gem install rails to install the popular Ruby on Rails web framework.
